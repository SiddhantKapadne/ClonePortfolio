# Case Study Image Structure

Store all case study visuals in this folder using one dedicated folder per case study.

## Convention

- `src/assets/case-studies/<case-study-name>/`
- Keep filenames descriptive and predictable.

## Current

- `chatbot/reference.png`

## Example for future case studies

- `src/assets/case-studies/payments/hero.png`
- `src/assets/case-studies/payments/flow-1.png`
- `src/assets/case-studies/healthcare/research-board.png`
